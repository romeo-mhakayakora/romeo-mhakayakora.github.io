05 // Curated Learning

"Old" Reinforcement Learning (Classical Foundations)

A rigorous study plan tracking the mathematical foundations of sequential decision making, covering tabular models, dynamic programming, and model-free exact solutions.

Module 01

Introduction to RL & MDPs

Agent-environment interaction loop, states, actions, rewards, transition dynamics, Markov property, and return/discounting formulations.

Module 02

Bellman Equations & Optimality

Value functions, action-value functions, Bellman expectation equations, Bellman optimality equations, and state-action value mappings.

Module 03

Dynamic Programming

Policy evaluation, policy iteration, value iteration, contraction mapping theorem, and generalized policy iteration (GPI).

Module 04

Model-Free Prediction

Monte Carlo learning, Temporal Difference learning, TD(0) prediction, and TD(λ) bootstrapping.

Module 05

Model-Free Control

On-policy tabular control (Sarsa), off-policy tabular control (Q-learning), and exploration/exploitation tradeoffs.

Module 06

Exploration Bandits & Linear Approximation

Multi-armed bandits, Upper Confidence Bound (UCB) algorithms, Thompson sampling, and linear value function approximation.
