Machine Learning Concepts

This page explains essential machine learning concepts, focusing on feature scaling techniques.

1. Normalization (Min‑Max Scaling)

Formula:

x_scaled = (x - min(x)) / (max(x) - min(x))

Normalization scales features to a range of [0, 1]. It is useful when you need bounded intervals, such as in image processing where pixel values are 0‑255, or when using algorithms that assume input features are on a similar scale (e.g., neural networks with sigmoid activation).

When to use:

  • When the data does not follow a Gaussian distribution.
  • When algorithms are sensitive to the scale of features (e.g., k‑nearest neighbors, support vector machines with RBF kernel).
  • When you need to preserve zero entries in sparse data? Actually normalization does not preserve zero; standardization does better with sparsity.

2. Standardization (Z‑Score Normalization)

Formula:

x_scaled = (x - μ) / σ

where μ is the mean and σ is the standard deviation.

Standardization transforms data to have zero mean and unit variance. It is appropriate when the data follows a normal distribution or when algorithms assume normality (e.g., linear regression, logistic regression, linear discriminant analysis).

When to use:

  • When features have varying scales and you want to give each equal importance.
  • When using algorithms that assume normally distributed inputs (e.g., PCA, LDA).
  • When dealing with outliers, as standardization is less affected by extreme values than normalization (though still impacted).

3. Comparison

Property Normalization (Min‑Max) Standardization (Z‑Score)
Range [0, 1] (-∞, ∞)
Mean Not zero (depends on distribution) 0
Standard deviation Not 1 1
Outlier sensitivity High Less sensitive
Preserves zero entries No No (unless mean is zero)
