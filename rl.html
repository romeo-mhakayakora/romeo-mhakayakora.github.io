Agentic RL 2026

OpenClaw-RL: Train Any Agent Simply by Talking

Yinjie Wang, Xuyang Chen, Xiaolong Jin, Mengdi Wang, Ling Yang (Princeton University)

Introduces a fully asynchronous reinforcement learning framework that optimizes agent policies in real-time during live deployment. Decouples serving, execution, reward judging, and training loops, leveraging scalar Process Reward Models and token-level Hindsight-Guided On-Policy Distillation.

RL Distillation 2025

On-Policy Distillation

Kevin Lu, Thinking Machines Lab

On-policy distillation optimizes student LLM policies by evaluating student rollouts against a fixed teacher using token-level reverse KL divergence. This eliminates compounding off-policy evaluation errors, providing a dense training signal that matches RL reasoning performance at a fraction of the compute cost.

RL for LLM & Agentic Alignment 2026

GrandCode: Achieving Grandmaster Level in Competitive Programming via Agentic Reinforcement Learning

Xiaoya Li, Xiaofei Sun, Guoyin Wang, Songqiao Su, Chris Shum, Jiwei Li (DeepReinforce Team)

Introduces GrandCode, a multi-agent reinforcement learning system that achieves grandmaster level in competitive programming by utilizing Agentic GRPO (Group Relative Policy Optimization) to coordinate solvers, hypothesis generators, and test verifiers.
