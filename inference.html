Hardware-Aware Runtimes & Quantization

Scaling deep learning models is only half the battle; serving them efficiently on heterogeneous hardware is where execution details matter. My work targets on-device and edge deployment frameworks. By optimizing memory layouts, kernels, WebGPU runtimes, and mapping execution graphs to edge NPUs and TPUs, we can run high-capability models within tight hardware and power envelopes.
