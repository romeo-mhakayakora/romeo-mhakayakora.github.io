LLM Inference Systems Syllabus

Module 01

Transformer Fundamentals

Attention mechanisms, encoder vs. decoder, tokenization, embeddings, positional encoding, and KV cache basics.

Module 02

Autoregressive Decoding

Prefill vs. decode phases, causal attention, greedy/beam search, top-k/top-p sampling, temperature, and speculative decoding.

Module 03

GPU & CUDA Basics

GPU architecture, CUDA kernels, memory hierarchy (HBM, SRAM, registers), kernel launches, tensor cores, and memory bandwidth limits.

Module 04

PyTorch Execution

Eager mode execution, compilation models (torch.compile), custom operators, dynamic shapes, and memory management inside PyTorch.

Module 05

Model Compression

FP16, BF16, INT8, INT4 quantization, weight-only and activation quantization, GPTQ, AWQ, GGUF, and KV cache quantization.

Module 06

Serving Architecture

API servers, OpenAI-compatible endpoints, FastAPI request lifecycle, batching mechanisms, and streaming responses.

Module 07

LLM Inference Engines

Comparative analysis of engines: vLLM, TensorRT-LLM, SGLang, Hugging Face TGI, LMDeploy, llama.cpp, Ollama, and ONNX Runtime GenAI.

Module 08

Scheduling

Continuous/iteration-level batching, request queues, paged attention algorithms, memory fragmentation, and KV cache management.

Module 09

KV Cache Optimization

Prefix caching, LMCache, cache reuse across requests, CPU/SSD offloading, disaggregated prefill, and cache eviction strategies.

Module 10

Performance Metrics

Time-to-First-Token (TTFT), Time-Per-Output-Token (TPOT), throughput/latency tradeoffs, GPU/memory utilization, concurrency, and tail latency.

Module 11

Distributed Inference

Tensor parallelism (TP), pipeline parallelism (PP), multi-GPU/multi-node scaling, and communication collectives (All-Reduce, All-Gather).

Module 12

Production Deployment

Autoscaling, load balancing, observability, Prometheus metrics, Grafana dashboards, tracing, fault tolerance, and cost optimization.
