Introduction to Actor-Critic

From REINFORCE to Actor-Critic: combining policy gradients with value-based learning.

Motivation

REINFORCE is an unbiased estimator of the policy gradient, but it suffers from high variance because it uses Monte Carlo returns from a single trajectory. The high variance makes learning slow and sample-inefficient.

Actor-Critic methods address this by introducing a critic — a value function that is learned alongside the policy (the actor) — to reduce variance while maintaining a reasonable bias.

The Actor-Critic Architecture

  • Actor ($\pi_\theta$): the policy network that decides which actions to take.
  • Critic ($V_\phi$ or $Q_\phi$): a value network that estimates the value function, providing a baseline or advantage estimate for the actor's updates.
Key idea: The critic bootstraps — it estimates the value of the current state using its own learned parameters, rather than waiting for the full Monte Carlo return. This introduces some bias but dramatically reduces variance, and the bias can be controlled by the choice of advantage estimator.

From REINFORCE to Actor-Critic

Recall the REINFORCE update:

$$\nabla_\theta J(\pi_\theta) \approx \sum_{t=0}^{T} R_t(\tau) \nabla_\theta \log \pi_\theta(a_t\mid s_t)$$

The return $R_t(\tau)$ is a random variable with high variance. In Actor-Critic, we replace $R_t(\tau)$ with an estimate that uses the critic:

$$\nabla_\theta J(\pi_\theta) \approx \sum_{t=0}^{T} \big(\text{advantage estimate}\big) \nabla_\theta \log \pi_\theta(a_t\mid s_t)$$

The simplest advantage estimate is the TD error:

$$\delta_t = r_t + \gamma V_\phi(s_{t+1}) - V_\phi(s_t)$$

where $V_\phi$ is the critic's estimate of the value function. The critic is trained to minimize the TD error (or a related loss), making it a more accurate value estimator over time.

Pseudocode Skeleton

Basic Actor-Critic loop (per step):
  1. Take action $a_t \sim \pi_\theta(\cdot\mid s_t)$, observe $r_t, s_{t+1}$.
  2. Compute TD error: $\delta_t = r_t + \gamma V_\phi(s_{t+1}) - V_\phi(s_t)$.
  3. Update actor: $\theta \leftarrow \theta + \alpha_\pi \delta_t \nabla_\theta \log \pi_\theta(a_t\mid s_t)$.
  4. Update critic: $\phi \leftarrow \phi + \alpha_V \delta_t \nabla_\phi V_\phi(s_t)$.

This is the simplest form. More sophisticated variants (A2C, GAE) use multi-step returns and more accurate advantage estimates, which we cover in the following sections.
